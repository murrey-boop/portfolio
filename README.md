## Usage

Simply open `index.html` in your web browser to view the portfolio website.

For deployment, you can host this on:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

## Customization

To personalize this portfolio:

1. Update the name and personal information in `index.html`
2. Add your own projects in the Projects section
3. Update contact information with your email, LinkedIn, and GitHub profiles
4. Modify colors in `styles.css` by changing the CSS variables in `:root`
5. Add your own profile image by replacing the placeholder
